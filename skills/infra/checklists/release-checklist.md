# Release Checklist

## Подготовка
- [ ] Все PR для релиза смержены
- [ ] Все тесты проходят на main
- [ ] CHANGELOG обновлён
- [ ] Версия в `pyproject.toml` / `package.json` обновлена
- [ ] Зависимости закреплены (lock-файл актуален)

## Staging
- [ ] Сборка образа успешна
- [ ] Деплой в staging выполнен
- [ ] Smoke-тесты пройдены
- [ ] Миграции БД применены без ошибок
- [ ] Ручная проверка критических сценариев

## Production
- [ ] Создан тег версии
- [ ] Деплой в production выполнен
- [ ] Мониторинг: метрики в норме, ошибок нет
- [ ] Логи чисты (нет новых ERROR)
- [ ] Пользователи не жалуются (первые 15 минут)

## Post-release
- [ ] CHANGELOG опубликован
- [ ] Уведомление команде отправлено
- [ ] Старый staging (если был) остановлен
- [ ] Артефакты предыдущей версии сохранены для отката

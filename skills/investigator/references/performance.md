# Performance — Профилирование и бенчмарки

## Профилирование
- Начни с высокоуровневого профилирования (cProfile, py-spy)
- Определи hot path — где проводится больше всего времени
- Спускайся на уровень функций/запросов

## Бенчмарки
- Измеряй до и после изменений
- Используй стабильное окружение (нет других нагрузок)
- Минимум 3 замера, бери медиану
- Учитывай cold start vs warm

## Поиск узких мест
- **CPU-bound**: профилирование, оптимизация алгоритмов, кэширование
- **I/O-bound**: асинхронность, connection pooling, батчинг
- **Memory-bound**: утечки памяти, большие аллокации, GC pressure
- **DB-bound**: EXPLAIN ANALYZE, индексы, N+1, денормализация

## Инструменты
- `cProfile` + `snakeviz` — профилирование CPU
- `py-spy` — профилирование без изменения кода
- `memory_profiler` — построчное потребление памяти
- `EXPLAIN ANALYZE` — анализ запросов PostgreSQL
- `httpx` / `locust` — нагрузочное тестирование API
